# Gebruiksaanwijzing

### opladen / vervangen batterijen

1. Schakelaar aan batterij afzetten
2. Schuif de batterij uit de robot en maak de connector los
3. Nu kan je hem in het laadstation steken

---

### draadloze communicatie
---
#### verbinding maken

1. Verbind je toestel met de Wifi "KOBE-LIJNVOLGER"
2. Surf in je browser naar "192.168.4.1"
3. Op de webpagina kan je de instellingen aanpassen

---

#### commando's

- start
- stop
- speed
- calibration speed
- set kp [0..]
- set ki [0..]
- set kd [0..]
- calibrate

---

### kalibratie

1. Plaats de robot op het midden van een lijn van het parcour
2. De robot zal er nu voor zorgen dat alle sensoren eens boven de zwarte lijn komen.
3. Tijdens deze bewegingen worden de waarden die de sensor meet verwerkt en wordt de sensor gekalibreerd.

---

### settings

De robot rijdt stabiel met volgende parameters
- Kp [0.10]
- ki [0.00]
- kd [0.00]
- speed [180 of minder]
- calibration speed [50]

---

### start/stop button
De knop start en stopt de robot. 
De knop bevindt zich vanboven op de robot.
